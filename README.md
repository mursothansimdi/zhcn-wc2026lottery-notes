# zhcn-wc2026lottery-notes

本仓库用于归档和发布多个独立的 HTML 页面，内容以信息整理与记录为主，不针对任何具体网站或服务。

## 项目简介

这是一个以 HTML 格式存放静态页面的归档仓库。页面内容主要涉及信息汇总与备注，适合需要以网页形式展示的长期存档场景。所有页面均为独立文件，不依赖后端服务，可直接通过浏览器打开查看。

## 目录说明

```
.
├── README.md          # 本文件
├── index.html         # 项目入口页面（可选）
└── pages/             # 存放各独立 HTML 页面
    ├── page-001.html
    ├── page-002.html
    └── ...
```

- `pages/` 目录下为各独立页面，文件名按顺序或主题命名。
- 如有索引页面，位于仓库根目录的 `index.html`。

## 页面归档说明

- 每个页面均为独立的 HTML 文件，包含完整的文档结构。
- 页面内容以客观信息整理为主，不含交互式表单或动态数据加载。
- 所有页面均使用纯前端技术（HTML + CSS）构建，无外部依赖（除非在页面内注明）。
- 页面可能引用本地资源（如图片、样式、脚本），均存放于对应页面所在目录。
